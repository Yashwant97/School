# School
This porject is about school management. this project can be used by authorities in school to make proper records and keep them safely.
Through this project a user can add, update, or delete the details for a student or teacher. He can also manage if a student has submitted his or her fees.

1.Download the project.
2.Open a project in Netbeans IDE.
3.Browse and open the project where you have downloaded the project.
